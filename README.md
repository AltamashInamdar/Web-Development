# HTML (HyperText Markup Language) – Structure Creates the basic structure of a website Defines headings, paragraphs, images, links, buttons, etc. Tells the browser what content to display

# CSS (Cascading Style Sheets) – Design Makes the website look attractive and styled ,Controls colors, fonts, spacing, and layout, Helps make the website responsive (mobile-friendly)

# JavaScript – Functionality Adds interactivity and dynamic behavior. Handles button clicks, form validation, animations. Updates content without reloading the page
